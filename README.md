# IT-Passport

## リポジトリ概要
ITパスポート取得の為の学習の軌跡を残す為。

### ITパスポートとは
ITパスポート(ｉパス)は、ITを利活用するすべての社会人・これから社会人となる学生が備えておくべき、ITに関する基礎的な知識が証明できる国家試験です。

### 資格選択理由
基本情報技術者取得の前段階として、ITパスポートでIT業界の共通的知識の習得をする為。
